# Show Go

## Purpose

 Help Users find multiple new music events with the help of they're spotify account preferences in any location of the users choosing!

## Built With

- HTML
- CSS
- JavaScript
- Bulma
- Moment.js
- Spotify server side API
- Ticketmaster server side API
